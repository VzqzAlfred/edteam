Estilos especiales dependiendo del contexto, posiciòn o estado.

:root

Sirve para declarar variables CSS


:hover

Se presenta cuando el usuario coloca el puntero sobre algun elemento, pero no necesariamente está activo

:active

Cubre el caso en que un elemento esta siendo activado por el usuario. Se usa más en <a>y<button> de HTML

El orden si importa porque va en cascada

  1. :link{...}
  2. :visited{...}
  3. :hover{...}
  4. :active{...}

:target

Permite tener interacción sin JavaScript. si existe alguno, con un id coincidente con el identificador de fragmentos de la URl del documento.

Cerrar

:nth-child :first-child :last-child

  • nth-child: Selecciona el elemento indicado que sea el enésimo hijo de su elemento padre.
  • first-child: Selecciona el elemento indicado que sea el primer hijo de su elemento padre.
  • last-child: Selec ciona el elemento indicado que sea el ultimo hijo de su elemento padre

:nth-of-type :first-of-type :last-of-type

  • :nth-of-type: Selecciona el elemento marcado que sea el enésimo hijo en su tipo sin importar la jerarquía dentro del elemento padre.
  • first-of-type: Selecciona el elemento indicado que sea el primer hijo en su tipo sin importar la jerarquía dentro del elemento padre.
  • last-of-type Selecciona el elemento indicado que sea el último hijo en su tipo sin importar la jerarquía dentro del elemento padre.

Ejejmplo interactivo

:not

Es una notación funcional que toma un selector simple X como argumento. Coincide con un elemento que no está representado por el argumento. X no debe contener otro selector de negación.

:empty

Corresponde a un elemento sin ningun nodo hijo. Sólo cuando hay nodos de elemento o texto con uno o más caracteres dentro del elemento ése se convierte en vacío.
